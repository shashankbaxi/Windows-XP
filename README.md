Windows XP Themed Portfolio
Overview

This project is a personal portfolio website designed with a Windows XP–inspired user interface, recreating the nostalgic desktop experience using modern web technologies. The portfolio blends retro aesthetics with contemporary front-end development practices to present personal information, skills, and projects in an interactive and visually distinctive format.

The application is entirely client-side and implemented using HTML, CSS, and JavaScript, without reliance on external frameworks.

Key Features

Windows XP–style desktop interface

Taskbar-inspired navigation layout

Window-based sections for content display

Interactive UI elements such as draggable or clickable panels

Responsive behavior adapted for different screen sizes

Lightweight, fast-loading implementation using vanilla technologies

Technologies Used

HTML5 – Semantic structure and layout

CSS3 – Styling, animations, and Windows XP–inspired visuals

JavaScript (Vanilla) – Interactivity, event handling, and UI logic

Design Inspiration

The design is inspired by the classic Microsoft Windows XP operating system, focusing on familiar UI components such as windows, buttons, icons, and taskbar interactions. The goal is to evoke nostalgia while maintaining usability and performance in modern browsers.

Setup Instructions

Clone or download the project repository.

Open the main HTML file in any modern web browser.

No additional dependencies or build steps are required.

Usage

Navigate the portfolio as you would a desktop environment.

Open different sections through window-style components.

Interact with UI elements using mouse or touch input.

Customization

You can personalize the portfolio by:

Updating text content within the HTML files

Modifying styles in the CSS files to change colors, fonts, or layout

Extending JavaScript logic to add animations or new interactions

Replacing icons and images to better reflect your personal branding
